## рҹҺҜ Executive Summary

**SolIQ** is a **Digital Energy-as-a-Service (EaaS) platform** that makes clean solar energy accessible to every Indian вҖ” whether they own their home or rent an apartment. By replacing the вӮ№3,00,000+ upfront cost of rooftop solar with a flexible monthly subscription starting at вӮ№499/month, SolIQ addresses the single biggest barrier to India's residential solar revolution.

The platform uses a **"Dual Funnel" approach** вҖ” one app, two distinct pathways вҖ” to serve both homeowners (On-Site Rental model with physical hardware) and apartment renters (Virtual Subscription model with cloud-based energy credits), ensuring 100% market capture across all residential segments.

---

## рҹ”ҙ The Problem

India has immense solar potential вҖ” but the current model is fundamentally broken for most urban residents.

### The вӮ№3,50,000 Wall

A standard 5 kW rooftop solar system requires a вӮ№3,50,000 upfront investment вҖ” an insurmountable barrier for the majority of Indian urban households. But the financial wall is only part of the problem:

| Challenge | Who It Affects | The Reality |
|---|---|---|
| рҹ’ё **Massive CapEx Requirement** | All homeowners | вӮ№3,50,000+ to install a standard 5 kW system |
| рҹҸў **Subsidy Ineligibility** | Renters & tenants | PM Surya Ghar Yojana explicitly excludes apartment renters |
| рҹ”‘ **No Roof Ownership** | 40%+ urban residents | Cannot install hardware without owning the roof |
| рҹ”§ **Maintenance Anxiety** | All solar owners | Ongoing servicing, repairs, and battery replacements |
| рҹ“ү **Fragmented Billing** | DISCOM consumers | Complex net-metering paperwork and delayed settlements |

> **The result**: Only a tiny fraction of India's solar-eligible households have adopted rooftop solar. SolIQ exists to change that.

### Path A: Homeowners вҖ” On-Site Rental Model
NBFCs fund the hardware. SolIQ manages the subscription. **Zero CapEx risk for the customer.**
- App schedules a drone or manual site survey
- Instant NBFC loan approval in-app
- Free hardware installation + lifetime blackout immunity
- Customer pays a predictable monthly lease (вӮ№1,199вҖ“вӮ№4,999/month)
- Margins jump to **80%+** after Year 5 once the loan is cleared

### Path B: Renters вҖ” Virtual Subscription Model
A **"Swiggy for Solar"** model вҖ” SolIQ buys wholesale energy from partner farms and sells digital energy credits directly to renters via DISCOM smart meter integration.
- App links to the user's existing DISCOM Smart Meter
- User selects a Virtual Energy Tier (70 / 150 / 320 units)
- Digital energy units are credited instantly to the meter
- Direct utility bill reduction вҖ” no physical installation required

## рҹҡҖ Key Differentiators & USPs

### 1. Smashes the CapEx Barrier
Drops the typical вӮ№3,00,000+ entry cost to **вӮ№0 upfront**. Solar is no longer a capital purchase вҖ” it's a utility subscription.

### 2. Bypasses Real Estate Hurdles
Virtual subscriptions eliminate the need for roof ownership or landlord approvals entirely. If you have an electricity meter, you can go solar.

### 3. Removes Maintenance Anxiety
All hardware maintenance, cleaning, repairs, insurance, and battery replacements are handled by SolIQ вҖ” automatically, via the digital platform.

### 4. The "Uber for Energy" Business Model
A smart-wallet ecosystem that unifies physical rooftop rentals (Path A) and virtual cloud energy credits (Path B) into a single platform вҖ” achieving 100% market capture.

### 5. Asset-Light Software Play
By using NBFC debt to fund hardware, SolIQ achieves **SaaS-level gross margins** (25вҖ“85%) while removing hardware debt and manufacturing risk from its own balance sheet.

### 6. Guaranteed Collection via Hardware-Link
For Path A, payment is physically linked to power delivery. Lapse in subscription вҶ’ system auto-throttles in <5 seconds. **Zero NPAs. Zero revenue leakage.**

**Layer 1 вҖ” The Edge Layer (Real-Time Data Capture)**
IoT Smart Sticks (ESP32-S3 + MAX485) connect to solar inverters via Modbus RS485, capturing real-time telemetry and transmitting via MQTT over WiFi or 4G LTE fallback. A built-in SD card provides a **zero-data-loss buffer** during connectivity outages. Built-in tamper sensors detect unauthorized bypass and instantly shut off power export.

**Layer 2 вҖ” The Cloud Layer (AI-Driven Analytics)**
AWS-hosted servers receive the MQTT data stream. A centralized **AI "brain"** (Python + TensorFlow) processes data for predictive energy analytics, virtual tier optimization, fault detection, and automated billing.

**Layer 3 вҖ” The Service Ecosystem (API & Access)**
The processed intelligence is exposed via a secure API gateway connecting to: Fintech shields (Razorpay/Stripe), DISCOM utility towers (virtual net-metering), and user-facing mobile and web applications.

## рҹ’° Business Model & Unit Economics

SolIQ operates two parallel revenue models вҖ” designed to maximize margin while minimizing balance sheet risk.

### рҹҸ  Model A: On-Site Rental (Hardware Model)

*NBFCs fund the hardware. SolIQ earns a recurring management spread.*

|---|---|---|---|---|---|
| 1 kW Starter | вӮ№1,199/mo | вӮ№850 | вӮ№150 | **вӮ№199 (16%)** | вӮ№1,049/mo |

> рҹ“Ҳ **Margins jump to 80%+** after Month 60 once hardware debt is fully amortized, generating high-margin recurring revenue through Year 20.

### рҹҸў Model B: Virtual Subscription (Software Model)

*A "Swiggy for Solar" marketplace вҖ” SolIQ buys wholesale solar, sells digital credits with a software margin.*

| Plan | Monthly Price | Units | Farm Cost | DISCOM Toll | **SolIQ Profit** |
|---|---|---|---|---|---|
| Mini Pack | вӮ№499/mo | 70 units | вӮ№315 | вӮ№49 | **вӮ№123 (25%)** |
| Standard Pack | вӮ№999/mo | 150 units | вӮ№675 | вӮ№105 | **вӮ№194 (20%)** |

> вҡЎ **The Master Rule**: In Rental, DISCOM gets вӮ№0 (Net Metering). In Virtual, DISCOM receives вӮ№0.70/unit as a 'Wheeling Charge' вҖ” keeping the utility aligned with SolIQ's growth.

**`backend/.env`**
```env

# AWS

MQTT_BROKER_HOST=localhost
MQTT_BROKER_PORT=1883
MQTT_TOPIC_ROOT=soliq/devices/

RAZORPAY_KEY_ID=rzp_live_xxxx
RAZORPAY_KEY_SECRET=your_secret

# Virtual Net-Metering
DISCOM_API_URL=https://api.discom.gov.in/v1
DISCOM_API_KEY=your_discom_key

# NBFC Integration
NBFC_API_URL=https://api.nbfc-partner.com
NBFC_API_KEY=your_nbfc_key
```

**`app/.env`**
```env
FIREBASE_API_KEY=your_firebase_key
GOOGLE_LENS_API_KEY=your_lens_key      # KYC face match
RAZORPAY_KEY_ID=rzp_live_xxxx
API_BASE_URL=https://api.soliq.in/v1
```

### 4. Start Full Stack with Docker

```bash
docker-compose up --build
```

Starts: FastAPI backend, TimescaleDB, Mosquitto MQTT broker, Redis cache, and Firebase emulators.

### 5. Development Mode

```bash
# Backend (from /backend)
pip install -r requirements.txt
uvicorn main:app --reload --port 8000

# Flutter App (from /app)
flutter pub get
flutter run

# MQTT Broker (local)
mosquitto -c mosquitto.conf
```

### 6. Flash Firestick Firmware (Path A Hardware)

```bash
# Install PlatformIO
pip install platformio

cd firmware/

# Configure device credentials
cp config.h.example config.h
# Edit config.h: WiFi SSID/password, MQTT broker address, device ID

# Build & flash to ESP32-S3
pio run --target upload --environment esp32s3

# Monitor serial output for debugging
pio device monitor --baud 115200
```
