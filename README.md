## 🎯 Executive Summary

**SolIQ** is a **Digital Energy-as-a-Service (EaaS) platform** that makes clean solar energy accessible to every Indian — whether they own their home or rent an apartment. By replacing the ₹3,00,000+ upfront cost of rooftop solar with a flexible monthly subscription starting at ₹499/month, SolIQ addresses the single biggest barrier to India's residential solar revolution.

The platform uses a **"Dual Funnel" approach** — one app, two distinct pathways — to serve both homeowners (On-Site Rental model with physical hardware) and apartment renters (Virtual Subscription model with cloud-based energy credits), ensuring 100% market capture across all residential segments.

---

## 🔴 The Problem

India has immense solar potential — but the current model is fundamentally broken for most urban residents.

### The ₹3,50,000 Wall

A standard 5 kW rooftop solar system requires a ₹3,50,000 upfront investment — an insurmountable barrier for the majority of Indian urban households. But the financial wall is only part of the problem:

| Challenge | Who It Affects | The Reality |
|---|---|---|
| 💸 **Massive CapEx Requirement** | All homeowners | ₹3,50,000+ to install a standard 5 kW system |
| 🏢 **Subsidy Ineligibility** | Renters & tenants | PM Surya Ghar Yojana explicitly excludes apartment renters |
| 🔑 **No Roof Ownership** | 40%+ urban residents | Cannot install hardware without owning the roof |
| 🔧 **Maintenance Anxiety** | All solar owners | Ongoing servicing, repairs, and battery replacements |
| 📉 **Fragmented Billing** | DISCOM consumers | Complex net-metering paperwork and delayed settlements |

> **The result**: Only a tiny fraction of India's solar-eligible households have adopted rooftop solar. SolIQ exists to change that.

### Path A: Homeowners — On-Site Rental Model
NBFCs fund the hardware. SolIQ manages the subscription. **Zero CapEx risk for the customer.**
- App schedules a drone or manual site survey
- Instant NBFC loan approval in-app
- Free hardware installation + lifetime blackout immunity
- Customer pays a predictable monthly lease (₹1,199–₹4,999/month)
- Margins jump to **80%+** after Year 5 once the loan is cleared

### Path B: Renters — Virtual Subscription Model
A **"Swiggy for Solar"** model — SolIQ buys wholesale energy from partner farms and sells digital energy credits directly to renters via DISCOM smart meter integration.
- App links to the user's existing DISCOM Smart Meter
- User selects a Virtual Energy Tier (70 / 150 / 320 units)
- Digital energy units are credited instantly to the meter
- Direct utility bill reduction — no physical installation required

## 🚀 Key Differentiators & USPs

### 1. Smashes the CapEx Barrier
Drops the typical ₹3,00,000+ entry cost to **₹0 upfront**. Solar is no longer a capital purchase — it's a utility subscription.

### 2. Bypasses Real Estate Hurdles
Virtual subscriptions eliminate the need for roof ownership or landlord approvals entirely. If you have an electricity meter, you can go solar.

### 3. Removes Maintenance Anxiety
All hardware maintenance, cleaning, repairs, insurance, and battery replacements are handled by SolIQ — automatically, via the digital platform.

### 4. The "Uber for Energy" Business Model
A smart-wallet ecosystem that unifies physical rooftop rentals (Path A) and virtual cloud energy credits (Path B) into a single platform — achieving 100% market capture.

### 5. Asset-Light Software Play
By using NBFC debt to fund hardware, SolIQ achieves **SaaS-level gross margins** (25–85%) while removing hardware debt and manufacturing risk from its own balance sheet.

### 6. Guaranteed Collection via Hardware-Link
For Path A, payment is physically linked to power delivery. Lapse in subscription → system auto-throttles in <5 seconds. **Zero NPAs. Zero revenue leakage.**

## 🏗️ Architecture

The SolIQ platform is built across three tightly integrated layers:

```
┌──────────────────────────────────────────────────────────────────────┐
│                       SOLIQ ARCHITECTURE                              │
├──────────────────────────────────────────────────────────────────────┤
│  SERVICE ECOSYSTEM  │  Fintech Shields      │  DISCOM Utility Towers │
│  (API & Access)     │  Razorpay / Stripe    │  Virtual Net-metering  │
│                     │  UPI Deep-link        │  User Applications     │
├──────────────────────────────────────────────────────────────────────┤
│  CLOUD LAYER        │  AWS / Firebase       │  AI Intelligence Brain │
│  (AI Analytics)     │  TimescaleDB          │  TensorFlow Predictive │
│                     │  Docker / K8s         │  Load & Billing Engine │
├──────────────────────────────────────────────────────────────────────┤
│  EDGE LAYER         │  ESP32-S3 + MAX485    │  Zero-Data-Loss Buffer │
│  (Real-Time Capture)│  ACS712 Sensors       │  SD Card + 4G Fallback │
│                     │  Modbus RS485         │  MQTT Protocol         │
├─────────────────────┴───────────────────────┴──────────────────────-┤
│                           SOLAR HARDWARE                              │
│             Path A: Physical Rooftop  │  Path B: Farm Aggregation    │
└──────────────────────────────────────────────────────────────────────┘
```

**Layer 1 — The Edge Layer (Real-Time Data Capture)**
IoT Smart Sticks (ESP32-S3 + MAX485) connect to solar inverters via Modbus RS485, capturing real-time telemetry and transmitting via MQTT over WiFi or 4G LTE fallback. A built-in SD card provides a **zero-data-loss buffer** during connectivity outages. Built-in tamper sensors detect unauthorized bypass and instantly shut off power export.

**Layer 2 — The Cloud Layer (AI-Driven Analytics)**
AWS-hosted servers receive the MQTT data stream. A centralized **AI "brain"** (Python + TensorFlow) processes data for predictive energy analytics, virtual tier optimization, fault detection, and automated billing.

**Layer 3 — The Service Ecosystem (API & Access)**
The processed intelligence is exposed via a secure API gateway connecting to: Fintech shields (Razorpay/Stripe), DISCOM utility towers (virtual net-metering), and user-facing mobile and web applications.

## 💰 Business Model & Unit Economics

SolIQ operates two parallel revenue models — designed to maximize margin while minimizing balance sheet risk.

### 🏠 Model A: On-Site Rental (Hardware Model)

*NBFCs fund the hardware. SolIQ earns a recurring management spread.*

| Plan | Monthly Sub | NBFC EMI | O&M Escrow | **SolIQ Profit** | Post Year-5 |
|---|---|---|---|---|---|
| 1 kW Starter | ₹1,199/mo | ₹850 | ₹150 | **₹199 (16%)** | ₹1,049/mo |
| 3 kW Standard | ₹2,999/mo | ₹1,850 | ₹300 | **₹849 (28%)** | ₹2,699/mo |
| 5 kW Premium | ₹4,999/mo | ₹3,860 | ₹450 | **₹689 (14%)** | ₹4,549/mo |

> 📈 **Margins jump to 80%+** after Month 60 once hardware debt is fully amortized, generating high-margin recurring revenue through Year 20.

### 🏢 Model B: Virtual Subscription (Software Model)

*A "Swiggy for Solar" marketplace — SolIQ buys wholesale solar, sells digital credits with a software margin.*

| Plan | Monthly Price | Units | Farm Cost | DISCOM Toll | **SolIQ Profit** |
|---|---|---|---|---|---|
| Mini Pack | ₹499/mo | 70 units | ₹315 | ₹49 | **₹123 (25%)** |
| Standard Pack | ₹999/mo | 150 units | ₹675 | ₹105 | **₹194 (20%)** |
| Family Pack | ₹1,999/mo | 320 units | ₹1,440 | ₹224 | **₹285 (14%)** |

> ⚡ **The Master Rule**: In Rental, DISCOM gets ₹0 (Net Metering). In Virtual, DISCOM receives ₹0.70/unit as a 'Wheeling Charge' — keeping the utility aligned with SolIQ's growth.

### Business Model Canvas (Summary)

| | **Rental Model (Path A)** | **Virtual Model (Path B)** |
|---|---|---|
| **Key Partners** | Local Installers, NBFC Lenders | Utility Providers, Virtual Net-Metering Enablers |
| **Customer Segment** | Urban Homeowners with roof access | Renters, Apartment Dwellers, Small Offices |
| **Value Proposition** | ₹0 Upfront, Blackout Immunity, Zero Maintenance | No Roof Needed, 100% Portable, Direct Bill Slashing |
| **Revenue Stream** | ₹1,199–₹4,999/month hardware lease | ₹499–₹1,999/month digital subscription |
| **Cost Structure** | Hardware Capital, Installation & Service | Digital Infrastructure, Utility Fees & Trading Costs |
| **Channels** | Direct Sales, Online Platform | Digital Marketing, Mobile App & Web Portal |

## 🚀 Getting Started

### Prerequisites

| Requirement | Version |
|---|---|
| Node.js | v18+ |
| Python | 3.10+ |
| Flutter SDK | 3.x |
| Docker & Docker Compose | Latest |
| Firebase CLI | Latest |
| PlatformIO | Latest (for firmware) |

### 1. Clone the Repository

```bash
git clone https://github.com/Anannya34/navsahyadri_hack
cd SOLIQ_App
```

### 2. Project Structure

```
SOLIQ_App/
├── readme.md
└── solar_mobile_app/          # Primary mobile application directory
    └── SOLIQ/                 # React Native (Expo) project root
        ├── assets/            # Static assets (images, fonts)
        ├── src/               # Application source code
        │   ├── components/    # Reusable UI components & features
        │   │   ├── ActionGrid.js
        │   │   ├── CarbonDashboard.js
        │   │   ├── KhataEnergyChart.js
        │   │   ├── MarketScreen.js
        │   │   └── ...
        │   └── screens/       # Full-screen views
        │       └── HomeScreen.js
        ├── App.js             # Main application entry point
        ├── app.json           # Expo project configuration
        ├── babel.config.js    # Babel configuration
        └── package.json       # Project dependencies & scripts
```

### 3. Configure Environment Variables

```bash
cp app/.env.example app/.env
cp backend/.env.example backend/.env
```

**`backend/.env`**
```env
# Firebase
FIREBASE_PROJECT_ID=your_project_id
FIREBASE_SERVICE_ACCOUNT=./firebase-key.json

# AWS
AWS_REGION=ap-south-1
TIMESCALEDB_URL=postgresql://user:pass@localhost:5432/soliq

# MQTT
MQTT_BROKER_HOST=localhost
MQTT_BROKER_PORT=1883
MQTT_TOPIC_ROOT=soliq/devices/

# Payments
RAZORPAY_KEY_ID=rzp_live_xxxx
RAZORPAY_KEY_SECRET=your_secret

# Virtual Net-Metering
DISCOM_API_URL=https://api.discom.gov.in/v1
DISCOM_API_KEY=your_discom_key

# NBFC Integration
NBFC_API_URL=https://api.nbfc-partner.com
NBFC_API_KEY=your_nbfc_key
```

**`app/.env`**
```env
FIREBASE_API_KEY=your_firebase_key
GOOGLE_LENS_API_KEY=your_lens_key      # KYC face match
RAZORPAY_KEY_ID=rzp_live_xxxx
API_BASE_URL=https://api.soliq.in/v1
```

### 4. Start Full Stack with Docker

```bash
docker-compose up --build
```

Starts: FastAPI backend, TimescaleDB, Mosquitto MQTT broker, Redis cache, and Firebase emulators.

### 5. Development Mode

```bash
# Backend (from /backend)
pip install -r requirements.txt
uvicorn main:app --reload --port 8000

# Flutter App (from /app)
flutter pub get
flutter run

# MQTT Broker (local)
mosquitto -c mosquitto.conf
```

### 6. Flash Firestick Firmware (Path A Hardware)

```bash
# Install PlatformIO
pip install platformio

cd firmware/

# Configure device credentials
cp config.h.example config.h
# Edit config.h: WiFi SSID/password, MQTT broker address, device ID

# Build & flash to ESP32-S3
pio run --target upload --environment esp32s3

# Monitor serial output for debugging
pio device monitor --baud 115200
```
